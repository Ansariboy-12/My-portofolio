💻 Tata Cara Membuat Website Sederhana

1️⃣ Buat Folder Proyek

Buat satu folder di komputer kamu, misalnya bernama “web-sederhana”. Di dalamnya nanti akan ada file HTML, CSS, dan JavaScript.

2️⃣ Buat File HTML

Buka Visual Studio Code → klik New File → simpan sebagai index.html. Lalu isi dengan struktur dasar berikut:

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Web Pertamaku</title>
</head>
<body>
  <h1>Halo Dunia!</h1>
</body>
</html>

3️⃣ Tambahkan CSS

Buat file baru bernama style.css di folder yang sama, lalu hubungkan ke HTML dengan tag berikut di bagian <head>:

<link rel="stylesheet" href="style.css">

Contoh isi CSS sederhana:

body {
  background-color: lightblue;
  text-align: center;
  font-family: Arial;
}

4️⃣ Tambahkan Interaksi dengan JavaScript

Buat file script.js dan hubungkan di akhir sebelum tag </body>:

<script src="script.js"></script>

Isi sederhana:

alert("Selamat datang di web pertamaku!");

5️⃣ Jalankan di Browser

Terakhir, klik dua kali file index.html atau buka dengan Chrome. 🎉 Kamu sudah berhasil membuat web sederhana!
